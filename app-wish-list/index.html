App Wish List

I created App Wish List in order to restore the missing wish list functionality that was previously included in the App Store prior to iOS 11. Apps can be added to the wish list from within the App Store using a share extension. The wish list also tracks price changes and is automatically updated using background fetch.

Technologies:
Swift, Share Extension, StoreKit, Background Fetch, Notifications
